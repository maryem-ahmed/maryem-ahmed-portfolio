# Maryem Ahmed — AI Engineer Portfolio

A modern, professional portfolio website for Maryem Ahmed, built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- Dark/light mode toggle
- Animated particle network hero background
- Interactive skill explorer with proficiency bars
- Filterable project showcase
- Animated timeline for experience
- Command palette quick navigation (⌘K)
- Scroll progress indicator
- Contact form with mailto integration
- Fully responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Theming:** next-themes

## Content Source

All portfolio content is sourced from `Maryem-Ahmed-AI.pdf` in the project root.
